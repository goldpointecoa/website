Resources Page Encryption Tool

How to use this tool

This tool helps you encrypt the contents of your resources page. You can either:

  1. Paste the HTML content from resources-original.html directly into the text area
  2. Or try to load the content automatically from resources-original.html (requires server)

After encrypting, you'll download a file that you can use to replace your existing resources.html file.

Auto Load

This method attempts to load resources-original.html from the server. It may not work if you're using the file locally or if the server isn't running.
