Encrypt Resources Page

Use this tool to encrypt the resources page with a password. This tool will automatically use the resources-original.html file as the source for encryption. The encrypted version will be downloaded, and you will need to replace the existing resources.html file with this downloaded file.

Instructions

  1. Enter the password you want to use (pre-filled with "goldpointe2025")
  2. Click "Encrypt Resources Page"
  3. A download will start automatically with the encrypted version of resources.html
  4. Upload this file to replace the existing resources.html in your website

Note: After encryption, make sure to test the resources page to confirm the password protection works correctly.
